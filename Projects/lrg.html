Star vs. Large Red Galaxy Two-Step Model

This project was focused on training a two-step model to classify unknown objects as a star or Large Red Galaxy and to estimate distances of the predicted LRGs from the Earth

Breakdown

1. Conduct univariate and bivariate exploratory analysis

Through this initial exploration, we found that there was an imbalance of classes in the data, so we decided to move forward by balancing the classes. We also found any correlations in our variables and selected features that would be most effective in our model

2. Train classification model

We then trained multiple classification models to best predict whether or not the unknown project was a star or a large red galaxy.

We trained a logistic regression model, decision trees, random forest, XGBoost, and KNN in which XGBoost had the lowest misclassification rate with random forest coming at a close second.

3. Train regression model

We then trained multiple regression models to best predict the distance between the earth and objects that were already classified as large red galaxies.

We trained a linear regression model, ridge regression, lasso regression, GAM, regression tree, random forest, gradient boosting, and KNN in which graident boosting had the lowest mean squared error with GAM coming at a close second.

4. Train two-step model model

We then needed to know which classification and regression models worked best together, so we trained multiple pairings. First, we predicted the unknown object, and given the objects that were predicted as large red galaxies, we then predicted its distance from the earth.
